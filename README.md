# AlfidoTech-Internship-Tasks
Contains all task submissions for Alfido Tech Web Development Internship.

# Front-End Mini Projects

This repository contains 4 small front-end projects built using HTML, CSS, JavaScript, and Bootstrap.

## Projects
1. **Simple Webpage** – Displays my name, a short bio, and an image.  
2. **Responsive Navigation Bar** – A responsive navbar that collapses on small screens.  
3. **Background Color Changer** – A button that changes the page background to a random color.  
4. **To-Do List** – A simple interactive to-do list to add and remove tasks.

## How to Use
- Open the respective HTML files in your browser:
  - `task1.html` → Simple Webpage  
  - `task2.html` → Navbar  
  - `task3.html` → Background Color Changer  
  - `task4.html` → To-Do List
